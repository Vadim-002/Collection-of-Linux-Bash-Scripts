# Collection-of-Linux-Bash-Scripts


# 系统初始化  

系统初始化，一共具备如下功能

- 判断是否是root用户在执行此操作
- 添加了 -v|--verbose 以及-h |--help 两个选项
- 关闭防火墙
- 关闭SElinux
- 判断系统版本
- 根据系统的版本执行不同的操作
- 安装了vim
- 配置VIM的默认设置
- 添加默认用户
- 配置yum源
- 安装常用软件 tree vsftpd telnet lftp
- 配置网络为自动获取IP并重启网络服务
